# 财经新闻爬虫 (Scrapy版) - 使用说明

## 一、环境准备

```bash
# 1. 安装 Python 3.9+
python --version

# 2. 安装依赖
pip install -r requirements.txt
```

## 二、快速开始

### 1. 初始化数据库
```bash
python run.py --init-db
```

### 2. 单次爬取新闻
```bash
python run.py --once
```
运行后自动从 **21 个** 财经网站抓取新闻并入库。

### 3. 爬取视频平台创作者
```bash
python run.py --video
```
从 B站、抖音搜索财经类视频创作者，保存UP主主页地址。

### 4. 全量爬取（新闻+视频）
```bash
python run.py --all
```

### 5. 查看统计
```bash
python run.py --stats
```

## 三、架构说明

本项目基于 **Scrapy 2.x** 框架重构：

```
pc/
├── scrapy.cfg                   # Scrapy 部署配置
├── run.py                       # 统一入口
├── config.py                    # 全局配置（关键词、来源列表）
├── requirements.txt             # 依赖清单
├── finance_crawler/             # Scrapy 项目
│   ├── items.py                 # Item 定义
│   ├── settings.py              # Scrapy 设置（并发、限速等）
│   ├── pipelines.py             # Item Pipeline（清理→去重→入库）
│   ├── middlewares.py           # 中间件（UA轮换）
│   ├── utils.py                 # 工具函数
│   └── spiders/
│       ├── eastmoney.py         # 东方财富
│       ├── sina_finance.py      # 新浪财经
│       ├── ...                  # 共 21 个新闻 Spider
│       ├── bilibili.py          # B站视频
│       └── douyin.py            # 抖音视频
├── models/
│   └── database.py              # 数据库模型
├── storage/
│   └── manager.py               # 存储管理器
└── export/
    └── exporter.py              # 数据导出
```

## 四、常用命令

| 命令 | 说明 |
|------|------|
| `python run.py --once` | 单次爬取全部 21 个新闻源 |
| `python run.py --once --source eastmoney,sina_finance` | 指定来源 |
| `python run.py --video` | 爬取B站+抖音创作者 |
| `python run.py --video --platform bilibili` | 只爬取B站 |
| `python run.py --all` | 同时爬取新闻+视频 |
| `python run.py --schedule` | 启动定时调度 |
| `python run.py --stats` | 查看统计信息 |
| `python run.py --search "降息"` | 关键词搜索新闻 |
| `python run.py --search-video "股市分析"` | 搜索视频创作者 |
| `python run.py --export json` | 导出新闻为 JSON |
| `python run.py --export-video json` | 导出视频创作者 JSON |

## 五、Scrapy 命令（高级）

```bash
# 列出所有 Spider
scrapy list

# 单独运行某个 Spider
scrapy crawl eastmoney

# 运行并导出到 JSON
scrapy crawl sina_finance -o output.json

# 查看 Spider 详情
scrapy genspider --list
```

## 八、环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `CRAWL_INTERVAL` | `30` | 定时爬取间隔（分钟） |
| `REQUEST_TIMEOUT` | `30` | 请求超时（秒） |
| `MAX_RETRIES` | `3` | 失败重试次数 |
| `FINANCE_CRAWLER_DB_URL` | `sqlite:///news.db` | 数据库地址 |

### 使用 MySQL
```bash
export FINANCE_CRAWLER_DB_URL="mysql+pymysql://用户名:密码@localhost:3306/finance_news"
```
