# OPC Nexus 品牌框架

> 基于品牌口号 **"洞察 · 链接 · 增长"** 梳理的平台内容架构、品牌视觉与体验原则。

---

## 一、品牌核心

| 维度 | 关键词 | 含义 |
|------|--------|------|
| **洞察** Insight | 信息、研究、判断 | 聚合新闻、政策、研报、视频，提供结构化摘要与行业分类。让用户在噪声中找到信号。 |
| **链接** Connect | 人脉、话题、社区 | 连接用户、作者、UP 主、机构、行业话题和线上线下活动。从孤立走向协作。 |
| **增长** Grow | 认知、业务、会员 | 通过高质量信息、专业讨论、活动和会员服务帮助用户获得认知与业务增长。 |

**品牌定位**：面向财经、产业、投资和企业经营人群的**专业内容与交流平台**。

**一句话**：判断不止于此。—— 不止于信息，还有连接；不止于观点，还有证据；不止于判断，还有同行。

## 二、内容体系（对应「洞察」）

```
┌─────────────────────────────────────────────────────┐
│                     OPC Nexus 内容生态                 │
├───────────────┬───────────────┬─────────────────────┤
│   资讯 News    │  政策 Policies │  洞察 Insights      │
│  快讯/要闻     │  法规/解读     │  专栏/研报/深度       │
├───────────────┴───────────────┴─────────────────────┤
│              视频 Videos（UP 主 / 机构）              │
│              社区 Community（讨论 / 问答）             │
│              需求广场 Demands（供需对接）              │
│              活动 Events（线上 / 线下）                │
└─────────────────────────────────────────────────────┘
```

### 内容分层

| 层级 | 内容类型 | 频率 | 用户价值 |
|------|---------|------|---------|
| **快** | 快讯、要闻、政策速递 | 实时/日更 | 第一时间知道发生了什么 |
| **深** | 专栏、研报、深度分析 | 周更/月更 | 理解背后的逻辑和影响 |
| **慢** | 视频、社区讨论、活动 | 持续 | 多角度验证、交流碰撞 |

### 信息组织原则

- **结构化摘要**：每篇文章/视频自动提取关键词、行业标签、核心摘要
- **信号筛选**：通过编辑精选 + 算法推荐，减少信息噪音
- **证据保留**：每条信息标注来源、时间、可信度，允许不同意见共存

### 连接路径

| 路径 | 实现方式 |
|------|---------|
| 人 → 人 | 关注、私信、@提及、社区互动 |
| 人 → 内容 | 订阅作者/UP主、收藏文章、关注话题 |
| 人 → 活动 | 活动报名、线上直播、线下聚会 |
| 人 → 需求 | 需求广场：发布需求、响应需求、合作对接 |

## 五、品牌视觉体系

### 色彩系统

| 角色 | 色值 | 名称 | 用途 |
|------|------|------|------|
| **主文字** | `#0b1d2c` | Ink | 标题、正文 |
| **次文字** | `#294457` | Ink Soft | 辅助说明、元信息 |
| **背景** | `#f6f4ef` | Paper | 页面底色 |
| **强调色** | `#b56a3b` | Copper | 链接、按钮、重要标记 |
| **辅助色** | `#3f7a76` | Teal | 数据卡片、分类标签、状态 |
| **分割线** | `rgba(11,29,44,.16)` | Line | 边界、分隔 |

### 品牌色灵感

品牌标识中使用的核心橙色系：

| 色值 | 应用场景 |
|------|---------|
| `#f16511` ~ `#f66e14` | Logo 主色调（活力橙） |
| `#fff8f4` ~ `#fff9f5` | 暖白底色（柔和对比） |

> Logo 的橙色比 CSS 中的 Copper (`#b56a3b`) 更鲜艳，建议在品牌物料（Logo、海报、社交媒体）中使用 Logo 本色，在 Web 界面中使用 Copper 保持阅读舒适度。

### 字体

| 用途 | 字体 | Fallback |
|------|------|---------|
| **标题** | Georgia / Noto Serif SC | serif |
| **正文** | Arial / Microsoft YaHei | sans-serif |
| **代码/数据** | ui-monospace | monospace |

### 设计原则

1. **克制**：大面积留白，信息密度适中，减少视觉疲劳
2. **层次**：标题(serif) vs 正文(sans-serif) 形成天然对比
3. **温度**：Copper 暖色点缀在冷色 Ink 基调中，专业但不冰冷
4. **质感**：Paper (`#f6f4ef`) 底色模仿纸张触感，区别于纯白网站
